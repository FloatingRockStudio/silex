# C++ API Reference

The packaged documentation site is generated with Doxygen during the standard CMake and release build flow.

For C++ consumers, the canonical public surface is the header set under `include/silex/`.

The most important entry points are:

- `silex::resolvers::GenericResolver`
- `silex::Box`
- `silex::SilexParseOptions`
- `silex::SilexContextResolve`
- `silex::SilexPathResolve`
- the interfaces under `include/silex/interfaces/`

Use the generated class, file, and namespace pages in the Doxygen HTML output for the authoritative reference.
